# StringConverterTool

.NET Core Global Tools for [String Converter](https://github.com/seayxu/StringConverter).

# Getting Started

- install
```
dotnet tool install -g stringconverter
```

- uninstall
```
dotnet tool uninstall -g stringconverter
```

- update
```
dotnet tool update -g stringconverter
```

- list
```
dotnet tool list -g
```
